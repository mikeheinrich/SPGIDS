Usage

Colors are available as CSS variables throughout the design system. Use them in your stylesheets like this:

:root { --color-brand: #d6002a; --color-primary: #006d89; --color-primary-hover: #00495b; /* ... */ } .my-element { background-color: var(--color-primary); color: var(--color-white); }
