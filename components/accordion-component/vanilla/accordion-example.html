Accordion Component Examples

Basic Accordion

Accordion components allow content to be expanded and collapsed. The header shows a chevron icon that rotates when opened.

Closed State

This is the accordion content. It can contain any HTML content including text, images, lists, and other components.

Open State

This accordion is open by default. The chevron icon points down when open. You can click the header to collapse it.
<div class="accordion" data-accordion>
  <button class="accordion-header">
    <span>Accordion Light</span>
    <span class="accordion-icon">
      <svg viewBox="0 0 14 14">...</svg>
    </span>
  </button>
  <div class="accordion-content">
    <div class="accordion-content-inner">
      Content goes here
    </div>
  </div>
</div>

Interactive Example

Click on the accordion headers below to expand and collapse them. Hover to see the hover state with bold text and red border.

This accordion demonstrates the expand/collapse functionality. When you click the header, the content area smoothly expands or collapses.

The chevron icon rotates from pointing right (closed) to pointing down (open).

On hover, the header text becomes bold and the top border turns red.

Note: This is a standalone example. In your project, make sure to include the design system base styles or define the required CSS variables.
