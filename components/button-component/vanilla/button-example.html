Button Component Examples

Primary Buttons

Primary buttons are used for the main call-to-action. They have a filled background with white text.

Default State

<button class="btn-primary">Button</button>

Size Variations

<button class="btn-primary btn-small">Small</button>
<button class="btn-primary">Default</button>
<button class="btn-primary btn-large">Large</button>

Disabled State

<button class="btn-primary" disabled>Disabled</button>

Full Width

<button class="btn-primary btn-full-width">Full Width Button</button>

With Icon

<button class="btn-primary">
  <span class="btn-icon">
    <svg viewBox="0 0 16 16">...</svg>
  </span>
  Button primary icon
</button>

Icon Only

<button class="btn-primary btn-icon-only" aria-label="Add">...</button>

Link buttons are used for text-based actions. They have no background or border, and the text darkens on hover.
