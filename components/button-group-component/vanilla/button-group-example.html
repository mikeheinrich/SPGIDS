Button Group Component Examples

Basic Button Group

Button groups act like tabs or radio buttons. Only one button can be active at a time. The active button has a gray background, red text, and a red top border.

Two Buttons

<div class="btn-group" data-button-group>
  <button class="btn-group-item active">button</button>
  <button class="btn-group-item">button</button>
</div>

Three Buttons

<div class="btn-group" data-button-group>
  <button class="btn-group-item active">button</button>
  <button class="btn-group-item">button</button>
  <button class="btn-group-item">button</button>
</div>
