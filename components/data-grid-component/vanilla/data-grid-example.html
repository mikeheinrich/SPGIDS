Grid with Custom Cell Rendering

Data grid with custom cell renderers for the Attention column with colored backgrounds.

initDataGrid({ containerId: 'grid-custom', columnDefs: [ { field: 'name', headerName: 'Name' }, { field: 'code', headerName: 'Code' }, { field: 'description', headerName: 'Description' }, { field: 'attention', headerName: 'Attention', cellRenderer: function(params) { const colors = { 'blue': 'rgba(0, 109, 137, 0.1)', 'red': 'rgba(143, 0, 28, 0.1)', 'green': 'rgba(0, 153, 0, 0.1)', 'white': 'transparent' }; const color = params.data?.color || 'white'; return '<div style="background-color: ' + colors[color] + '; padding: 4px; border-radius: 2px;">' + params.value + '</div>'; } }, { field: 'values', headerName: 'Values' } ], rowData: [ { name: 'Value', code: 'Value', description: 'Value', attention: 'Value', values: 'Value', color: 'blue' }, { name: 'Value', code: 'Value', description: 'Value', attention: 'Value', values: 'Value', color: 'white' }, { name: 'Value', code: 'Value', description: 'Value', attention: 'Value', values: 'Value', color: 'red' }, { name: 'Value', code: 'Value', description: 'Value', attention: 'Value', values: 'Value', color: 'green' }, { name: 'Value', code: 'Value', description: 'Value', attention: 'Value', values: 'Value', color: 'white' } ], height: 400 });

Grid with Pagination

Data grid with pagination enabled for large datasets.

initDataGrid({ containerId: 'grid-pagination', columnDefs: [ { field: 'name', headerName: 'Name' }, { field: 'code', headerName: 'Code' }, { field: 'description', headerName: 'Description' }, { field: 'attention', headerName: 'Attention' }, { field: 'values', headerName: 'Values' } ], rowData: generateLargeDataset(100), // 100 rows enablePagination: true, paginationPageSize: 10, height: 400 });

API Demo

Interact with the data grid programmatically using the returned API object.

Select rows and click the buttons above to interact with the API.
// Initialize with API reference const gridApi = initDataGrid({ containerId: 'grid-api', columnDefs: [...], rowData: [...] }); // Get selected rows const selected = gridApi.getSelectedRows(); // Select all rows gridApi.selectAll(); // Deselect all rows gridApi.deselectAll(); // Size columns to fit gridApi.sizeColumnsToFit();

Note: This example uses AG-Grid Community Edition v32.3.3 from local files in vendor/ag-grid/.
