Dropdown Component Examples

Default Dropdown

Standard dropdown with border and background. Single-select functionality.

initDropdown({ containerId: 'dropdown-default', variant: 'default', placeholder: 'Select an option', items: ['List item', 'List item', 'List item', 'List item', 'List item'], onSelect: function(value) { console.log('Selected:', value); } });

Light Dropdown

Minimal dropdown with blue text, no border or background.

initDropdown({ containerId: 'dropdown-light', variant: 'light', placeholder: 'Selected item', items: ['List item', 'List item', 'List item', 'List item', 'List item'], onSelect: function(value) { console.log('Selected:', value); } });

Multi-select Dropdown

Dropdown with checkboxes for multiple selection. Selected items display as chips with "+ X more" format.

initDropdown({ containerId: 'dropdown-multi', variant: 'multi-select', placeholder: 'Select items', items: ['Label', 'Label', 'Label'], selectedItems: ['Item One'], onSelect: function(value) { console.log('Selected:', value); }, onDeselect: function(value) { console.log('Deselected:', value); }, onChange: function(selectedItems) { console.log('All selected:', selectedItems); } });

Note: This is a standalone example. In your project, make sure to include the design system base styles or define the required CSS variables.
