# Footer Component - Vanilla Implementation

Vanilla JavaScript/CSS implementation of the footer component.

## Files

- `footer-component.css` - Complete footer styles
- `footer-component.js` - Footer functionality
- `footer-example.html` - Standalone example

## Usage

### Include Styles and Scripts

```html
<!-- Include design system base styles -->
<link rel="stylesheet" href="../../../../styles/base.css">

<!-- Include component base styles -->
<link rel="stylesheet" href="../base/base.css">

<!-- Include vanilla implementation -->
<link rel="stylesheet" href="footer-component.css">
<script src="footer-component.js"></script>
```

### HTML

```html
<div id="footer-container"></div>
```

The footer auto-initializes when the page loads.

## Dependencies

- Design system base styles (`styles/base.css`)
- Component base styles (`base/base.css`)
