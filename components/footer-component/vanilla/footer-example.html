Footer Component Example

Features

The footer component provides a collapsible footer with:

  • Copyright text (auto-updates with current year)
  • Footer links (Terms, Privacy, Customer Care)
  • S&P Global logo
  • Smooth toggle animation
  • Fixed position at bottom
  • Auto-initialization (no configuration needed)

Usage

Click the chevron button at the bottom to expand/collapse the footer.

Basic Implementation

<!-- Add container --> <div id="footer-container"></div> <!-- Include CSS --> <link rel="stylesheet" href="components/footer-component/footer-component.css"> <!-- Include JavaScript --> <script src="components/footer-component/footer-component.js"></script> <!-- Footer auto-initializes, no configuration needed -->

Important Notes

  • Fixed Position: The footer is fixed at the bottom. Add margin-bottom: 96px; to your main content.
  • Collapsed by Default: Footer starts collapsed (48px tall). Click to expand (96px tall).
  • Auto-Initialization: Footer automatically initializes when the page loads.
  • Copyright Year: Automatically updates to current year using JavaScript.
  • Responsive: Layout adapts for mobile devices.
