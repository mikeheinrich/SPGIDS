Header Component Example

Features

The header component provides a fixed navigation bar with:

  • S&P Global logo (clickable, links to home)
  • Product name display
  • Notification bell with badge counter
  • Help/question icon
  • User avatar with initials
  • Slide-out panels for user menu, help, and notifications

Usage

Click on the notification bell, help icon, or user avatar to see the slide-out panels in action.

Basic Implementation

<!-- Add containers --> <div id="header-container"></div> <div id="header-panels-container"></div> <!-- Include CSS --> <link rel="stylesheet" href="components/header-component/header-component.css"> <!-- Include JavaScript --> <script src="components/header-component/header-component.js"></script> <!-- Initialize --> <script> initHeader({ productName: 'My Product', userInitials: 'JD', notificationCount: 3, userName: 'John Doe', userEmail: 'john.doe@company.com' }); </script>

Configuration Options

Option Type Default Description
productName string '' Product name displayed next to logo
userInitials string 'U' User avatar initials (2 characters)
notificationCount number 0 Number of notifications (badge appears if > 0)
userName string '' User's full name
userEmail string '' User's email address

Important Notes

  • Fixed Position: The header is fixed at the top. Add margin-top: 52.5px; to your main content.
  • Z-index: Header has z-index 1000, panels have z-index 999, overlay has z-index 998.
  • Responsive: Product name is hidden on screens smaller than 768px.
  • Keyboard Support: Press Escape to close panels, Enter/Space on user avatar to open user panel.
