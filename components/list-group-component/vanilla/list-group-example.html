List Group Component Examples

Basic List Group

A simple list of selectable items. Click on any item to select it. The selected item will show a red accent bar at the top.

    <nav class="list-group" id="list-group-basic">
      <ul class="list-group-list" id="listGroupListBasic"></ul>
    </nav>

    <script>
      initListGroup({
        containerId: 'list-group-basic',
        listId: 'listGroupListBasic',
        items: [
          { id: '1', name: 'Item One' },
          { id: '2', name: 'Item Two' },
          { id: '3', name: 'Item Three' }
        ]
      });
    </script>
    Selected: None

    List Group with Initial Selection

    A list group with a pre-selected item.

      initListGroup({
        containerId: 'list-group-initial',
        listId: 'listGroupListInitial',
        items: [
          { id: '1', name: 'First Item' },
          { id: '2', name: 'Second Item' },
          { id: '3', name: 'Third Item' }
        ],
        initialSelection: '2'
      });

      List Group with Disabled Items

      Some items can be disabled and will not be selectable.

        initListGroup({
          containerId: 'list-group-disabled',
          listId: 'listGroupListDisabled',
          items: [
            { id: '1', name: 'Enabled Item' },
            { id: '2', name: 'Disabled Item', disabled: true },
            { id: '3', name: 'Another Enabled Item' }
          ]
        });

          List Group with Selection Callback

          A list group that triggers a callback function when an item is selected.

            initListGroup({
              containerId: 'list-group-callback',
              listId: 'listGroupListCallback',
              items: [
                { id: '1', name: 'Option A' },
                { id: '2', name: 'Option B' },
                { id: '3', name: 'Option C' }
              ],
              onItemSelect: function(id, item) {
                console.log('Selected:', id, item.name);
                alert('You selected: ' + item.name);
              }
            });
