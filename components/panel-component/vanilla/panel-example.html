Modal Component Examples

Basic Modal

Basic modal with header (title and close button), divider, and content area. Click the button to open the modal.

<div class="modal-overlay" id="basicModal">
  <div class="modal">
    <div class="modal-header">
      <h2 class="modal-title">Panel title</h2>
      <button class="modal-close" aria-label="Close modal">...</button>
    </div>
    <div class="modal-content">
      <p>Modal content here</p>
    </div>
  </div>
</div>

Programmatic Modal

Create and open a modal programmatically using JavaScript.

const modal = createModal({
  title: 'Panel title',
  content: '<p>Modal content here</p>',
  onOpen: function() { console.log('Modal opened'); },
  onClose: function() { console.log('Modal closed'); }
});
openModal(modal);

Modal with Custom Content

Modal can contain any HTML content.

<div class="modal-overlay" id="customModal">
  <div class="modal">
    <div class="modal-header">...</div>
    <div class="modal-content">
      <!-- Any HTML content -->
    </div>
  </div>
</div>
