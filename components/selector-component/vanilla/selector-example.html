Selector Component Examples

Checkbox Group

Checkboxes allow users to select one or more options from a list. Selected checkboxes display a red checkmark.

Default State

<div class="selector-item"> <input type="checkbox" id="checkbox1" class="selector-checkbox" checked /> <label for="checkbox1">Label</label> </div>

Disabled State

<input type="checkbox" class="selector-checkbox" disabled />

Radio Group

Radio buttons allow users to select a single option from a list. Selected radio buttons display a red filled circle.

Default State

<div class="selector-item"> <input type="radio" id="radio1" name="radio-group" class="selector-radio" checked /> <label for="radio1">Label</label> </div>

Disabled State

<input type="radio" class="selector-radio" disabled />

Switch

Switches allow users to toggle an option on or off. The on state uses primary blue, and the off state uses gray.

Default State

<div class="selector-item"> <input type="checkbox" id="switch1" class="selector-switch" checked /> <label for="switch1">Label</label> </div>

Disabled State

<input type="checkbox" class="selector-switch" disabled />

Note: This is a standalone example. In your project, make sure to include the design system base styles or define the required CSS variables.
