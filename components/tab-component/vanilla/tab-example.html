Tab Component Examples

Basic Usage

The tab component provides a tabbed interface for organizing content into multiple panels.

<!-- Include CSS --> <link rel="stylesheet" href="components/tab-component/base/base.css"> <link rel="stylesheet" href="components/tab-component/vanilla/tab-component.css"> <!-- Include JavaScript --> <script src="components/tab-component/vanilla/tab-component.js"></script> <!-- Container --> <div id="tabs-container"></div> <!-- Initialize --> <script> initTabs({ containerId: 'tabs-container', tabs: [ { id: 'tab1', label: 'Tab 1', content: '<p>Content for tab 1</p>' }, { id: 'tab2', label: 'Tab 2', content: '<p>Content for tab 2</p>' }, { id: 'tab3', label: 'Tab 3', content: '<p>Content for tab 3</p>' } ] }); </script>

Note: This is a standalone example. In your project, make sure to include the design system base styles or define the required CSS variables.
