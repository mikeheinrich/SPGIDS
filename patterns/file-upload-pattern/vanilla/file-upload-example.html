File Upload Pattern Examples

Vertical Layout (Default)

The file list appears below the drop zone. This is ideal for narrow containers or when you expect a longer list of files.

<div id="upload-vertical"></div> <script> initFileUpload({ containerId: 'upload-vertical', layout: 'vertical', dropzoneText: 'Drag and drop file to upload', acceptedFormats: 'Accepted formats: pdf, doc, docx', onFilesChange: function(files) { console.log('Files:', files); } }); </script>

Horizontal Layout

The file list appears to the right of the drop zone. This is ideal for wider containers where you want to utilize horizontal space.

<div id="upload-horizontal"></div> <script> initFileUpload({ containerId: 'upload-horizontal', layout: 'horizontal', dropzoneText: 'Drag and drop file to upload', acceptedFormats: 'Accepted formats: pdf, doc, docx' }); </script>

Image Upload

Configure the file upload to accept only image files with custom validation settings.

<div id="upload-images"></div> <script> initFileUpload({ containerId: 'upload-images', layout: 'vertical', dropzoneText: 'Drag and drop images to upload', acceptedFormats: 'Accepted formats: jpg, png, gif, webp', accept: '.jpg,.jpeg,.png,.gif,.webp,image/*', maxFileSize: 5 * 1024 * 1024, // 5MB maxFiles: 5, browseButtonText: 'Select images', fileListTitle: 'Selected images' }); </script>

Single File Upload

Restrict to uploading only a single file at a time.

<div id="upload-single"></div> <script> initFileUpload({ containerId: 'upload-single', layout: 'vertical', dropzoneText: 'Drag and drop a file to upload', acceptedFormats: 'Accepted formats: any', accept: '*', multiple: false, maxFiles: 1, browseButtonText: 'Select file', fileListTitle: 'Selected file' }); </script>

API Demo

Interact with the file upload programmatically using the returned API object.

Upload some files and click the buttons above to interact with the API.
// Initialize with API reference const uploader = initFileUpload({ containerId: 'upload-api-demo', onFilesChange: function(files) { console.log('Files changed:', files.map(f => f.name)); } }); // Get current files const files = uploader.getFiles(); // Clear all files uploader.clearFiles(); // Toggle disabled state uploader.setDisabled(true); uploader.setDisabled(false);

With Initial Files

Initialize the upload with pre-existing files (useful for edit scenarios).

<div id="upload-initial"></div> <script> initFileUpload({ containerId: 'upload-initial', layout: 'horizontal', initialFiles: [ { name: 'existing-document.pdf', size: 1024000 }, { name: 'report-2024.docx', size: 512000 } ] }); </script>

Note: This is a standalone example. In your project, make sure to include the design system base styles or define the required CSS variables.
