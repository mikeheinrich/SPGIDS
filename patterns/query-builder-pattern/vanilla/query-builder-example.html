Query Builder Pattern Example

A 3-column pattern for building queries by selecting attributes across multiple categories. The left column uses the List Group Component for category navigation, the center column shows selectable attributes with search filtering, and the right column displays a summary of selected items.

Left Column Category navigation using the List Group component with red accent bar for active state
Center Column Attribute selection with search filtering using the Search component
Right Column Summary of selections grouped by category with removable tags
