Right Toolbar Pattern Examples

Basic Actions

A simple toolbar with single-action items that trigger click handlers.

Tool name (Share)

Functions are handled in a flyout panel

<div id="toolbar-basic"></div> <script> initRightToolbar({ containerId: 'toolbar-basic', demoMode: true, items: [ { id: 'share', label: 'Share', icon: 'icon-content-share-arrow', type: 'action', onClick: function(item, id) { alert('Share action clicked!'); } }, { id: 'export', label: 'Export', icon: 'icon-content-download', type: 'action', onClick: function(item, id) { alert('Export action clicked!'); } }, { id: 'delete', label: 'Delete', icon: 'icon-content-delete', type: 'action', onClick: function(item, id) { alert('Delete action clicked!'); } } ] }); </script>

Actions with Panels

Toolbar items that open flyout panels when clicked. Panels appear to the left of the toolbar.

<div id="toolbar-panels"></div> <script> initRightToolbar({ containerId: 'toolbar-panels', demoMode: true, items: [ { id: 'share-panel', label: 'Share', icon: 'icon-content-share-arrow', type: 'panel', panelTitle: 'Share Options', panelContent: ` <div class="panel-content"> <h3>Share Options</h3> <p>Choose how you want to share this item.</p> <!-- Add your panel content here --> </div> ` }, { id: 'export-panel', label: 'Export', icon: 'icon-content-download', type: 'panel', panelTitle: 'Export Options', panelContent: ` <div class="panel-content"> <h3>Export Options</h3> <p>Select export format and options.</p> </div> ` } ] }); </script>

Actions with Modals

Toolbar items that open modal dialogs when clicked.

<div id="toolbar-modals"></div> <script> initRightToolbar({ containerId: 'toolbar-modals', demoMode: true, items: [ { id: 'delete-modal', label: 'Delete', icon: 'icon-content-delete', type: 'modal', modalTitle: 'Confirm Delete', modalContent: ` <div class="modal-content"> <h3>Are you sure?</h3> <p>This action cannot be undone.</p> <!-- Add your modal content here --> </div> ` } ] }); </script>

Mixed Actions (Actions, Panels, and Modals)

A toolbar combining all three types of actions.

<div id="toolbar-mixed"></div> <script> initRightToolbar({ containerId: 'toolbar-mixed', demoMode: true, items: [ { id: 'share', label: 'Share', icon: 'icon-content-share-arrow', type: 'action', onClick: function() { console.log('Share clicked'); } }, { id: 'export', label: 'Export', icon: 'icon-content-download', type: 'panel', panelTitle: 'Export Options', panelContent: '<div>Export panel content</div>' }, { id: 'delete', label: 'Delete', icon: 'icon-content-delete', type: 'modal', modalTitle: 'Confirm Delete', modalContent: '<div>Are you sure?</div>' } ] }); </script>

Full Example (Matching Design)

A complete example matching the provided design with Share, Export, Delete, and Swan actions.

<div id="toolbar-full"></div> <script> initRightToolbar({ containerId: 'toolbar-full', demoMode: true, items: [ { id: 'share', label: 'Share', icon: 'icon-content-share-arrow', type: 'panel', panelTitle: 'Share', panelContent: ` <div class="panel-content"> <h3>Share Options</h3> <p>Choose how to share this item.</p> </div> ` }, { id: 'export', label: 'Export', icon: 'icon-content-download', type: 'panel', panelTitle: 'Export', panelContent: ` <div class="panel-content"> <h3>Export Options</h3> <p>Select export format.</p> </div> ` }, { id: 'delete', label: 'Delete', icon: 'icon-content-delete', type: 'modal', modalTitle: 'Delete Item', modalContent: ` <div class="modal-content"> <h3>Are you sure?</h3> <p>This action cannot be undone.</p> </div> ` }, { id: 'swan', label: 'Swan', icon: 'icon-content-plus', type: 'action', onClick: function() { alert('Swan action triggered!'); } } ] }); </script>

Note: This is a standalone example. In your project, make sure to include the design system base styles and modal component styles.
