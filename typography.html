Font Family

Our primary font is Akkurat LL with Arial as the fallback. The font stack is:

font-family: 'Akkurat LL', -apple-system, BlinkMacSystemFont, 'Segoe UI', Arial, sans-serif;

Usage

Typography styles are available as CSS classes and variables:

/* Using CSS classes */ <h1 class="text-1">Heading</h1> <p class="text-paragraph">Body text</p> /* Using CSS variables */ .my-heading { font-size: var(--text-1); font-family: var(--font-primary); color: var(--color-gray-5); }
